A sample WordPress plugin for development purposes.

== Description ==

This is a sample WordPress plugin that demonstrates:

* Basic plugin structure
* Admin settings page
* Custom shortcodes
* CSS and JavaScript enqueuing
* Proper WordPress coding standards

Use this as a starting point for your custom WordPress plugins.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/my-custom-plugin` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->My Custom Plugin screen to configure the plugin

== Frequently Asked Questions ==

= How do I use the shortcode? =

Use `[my_custom_shortcode]` in any post or page. You can also add a custom message: `[my_custom_shortcode message="Your custom message"]`

= Where are the plugin settings? =

Go to Settings > My Custom Plugin in your WordPress admin dashboard.
